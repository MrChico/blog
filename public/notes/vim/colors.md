![Monokai](/images/notes/vim/monokai.png)

## Introduction

I started exploring how to make a theme for [Oni](https://github.com/onivim/oni). I wanted to make 
the sublime inspired monokai theme. 


###  How

In Oni a theme is defined as below:

```bash
./theme-monokai
├── colors
│   ├── monokai.json
│   └── monokai.vim
├── install.sh
└── package.json
```
